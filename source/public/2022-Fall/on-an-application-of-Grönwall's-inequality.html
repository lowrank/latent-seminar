    on an application of Grönwall's inequality

    Sep 25, 20226 min read

    • math-AP
    • math-CA

    Introduction

    We start with a simple application of Grönwall’s inequality.

    Grönwall's inequality

    If the following relation holds

    0≤f(x)≤C∫0x​f(s)ds,x∈[0,A],

    then f(x)=0.

    The traditional way (of course a nice one) is letting

    h(x)=e−Cx∫0x​f(s)ds,

    and utilize the relation

    h′(x)=e−Cx(f(x)−C∫0x​f(s)ds)≥0.

    Here, we attempt with an alternative way. We define the integral operator A:L+∞​[0,A]↦L+∞​[0,A] by the following:

    Af:=C∫0x​f(s)ds.

    Then A is a positive and compact operator (why?). According to the Krein-Rutman theorem1, if its spectral radius r(A) is a positive eigenvalue, and we must have the corresponding eigenfunction ϕ(x)>0 strictly positive. However, the definition of A implies that Aϕ(0)=0, which gives a contradiction. Therefore, the spectral radius r(A)=0.

    Observe that 0≤f(x)≤Af(x) implies 0≤f(x)≤Anf(x) (why?), the Gelfand’s formula implies ∥Ak∥op​→0 as k→∞, hence f(x)=0 by taking n→∞.

    Extension

    If there is another operator B that commutes with A, then the spectral radius r(A+B) can be estimated. We consider the abstract problem as follows.

    Extension of Grönwall's inequality

    Let B be a linear positive operator with r(B)<1 that commutes with A, and

    0≤f(x)≤Af+Bf,

    then f(x)=0.

    The commutativity implies an estimate r(A+B)≤r(A)+r(B)<1 (why?). Then the same argument holds since (A+B)k→0 as k→∞.

    This conclusion seems somewhat trivial. Let us consider an immediate application in transport equation.

    Background of transport equation

    The transport equation describes the dynamics of radiative particles interacting with the environment (absorption, scattering, etc.). For instance, supposing the medium is homogeneous, the governing equation can be written in the following form:

    v⋅∇u(x,v)+σa​uu∣Γ−​​​=σs​(Ku−u) in D×Sd−1,=h(x,v),​

    where K is the scattering operator, which represents the probability of scatter events that change direction v′ to the direction v. The function h(x,v) is the source defined on the incoming boundary set:

    Γ−​={(x,v)∈∂D×Sd−1∣v⋅n(x)<0}.

    Cone-beam source

    The cone-beam source function means h(x,v) is quite focusing. The precise definition is the following.

    Cone-beam source

    If h(x,v) satisfies that

    • h is non-negative;
    • There exists a set V⊂Sd−1 that supph⊂∂D×V∩Γ−​.

    Then h is called a cone-beam source.

    Intuitively, this means the source is one-way dominated (like a laser) and only supported on a certain subset of Γ−​. In a special case that σs​=0, we find the solution can be solved directly.

    u(x,v)=h(x−τ−​(x,v)v,v)E(x,x−τ−​(x,v)v),E(x,x−sv)=exp(−∫0s​σa​(x−tv)dt),

    where τ−​(x,v) denotes the distance from x to the boundary following −v direction.

    Nonlinearity

    In a practical scenario, σa​ may depend on the solution’s flux (e.g., multi-photon absorption), therefore we obtain a nonlinear equation

    u(x,v)=h(x−τ−​(x,v)v,v)exp(−∫0τ−​(x,v)​σa​(x−tv,⟨u⟩)dt),⟨u⟩=∫Sd−1​u(x,v)dv,

    where dv represents the usual probability measure on the sphere. Here, we need some continuity assumption.

    Assumption on Lipschitz continuity

    ∣σa​(x,⟨u⟩)−σa​(x,⟨w⟩)∣≤L∣⟨u⟩(x)−⟨w⟩(x)∣

    Let us first assume the existence of the solution and focus on the uniqueness. If there exists a solution w different from u, then we have:

    u−w=h(x−τ−​(x,v)v,v)[exp(−∫0τ−​(x,v)​σa​(x−tv,⟨u⟩)dt)−exp(−∫0τ−​(x,v)​σa​(x−tv,⟨w⟩)dt)].

    Lemma

    If a,b≥c≥0, then ∣e−a−e−b∣≤e−c∣a−b∣.

    Let f:=⟨u−w⟩, using this simple lemma, and integrate over Sd−1, there exists a constant C>0 that

    ∣f(x)∣≤h∫V​∫0τ−​(x,v)​∣σa​(x−tv,⟨u⟩)−σa​(x−tv,⟨w⟩)∣dtdv≤C∫V​∫0τ−​(x,v)​∣f(x−tv)∣dtdv,

    where h=supΓ−​​h and C is a constant. This inequality is just an analog of the previous Grönwall’s inequality, which has zero spectral radius. Thus, we must have f(x)=0, which proves the uniqueness.

    Remark

    The general uniqueness can be proved in a different flavor, but it requires slightly more restrictive dependence of σa​ on ⟨u⟩. For scatter-free medium plus a cone-beam source, it only needs Lipschitz continuity.

    Isotropic Scattering

    Once the scattering is present, the uniqueness is slightly more challenging (there is an alternative way to prove this), assume that σs​ is a positive constant, then we can represent the solution by (let σt​:=σa​+σs​)

    u(x,v)=​h(x−τ−​(x,v)v,v)exp(−∫0τ−​(x,v)​σt​(x−tv,⟨u⟩)dt)+∫0τ−​(x,v)​exp(−∫0s​σt​(x−tv,⟨u⟩)dt)σs​⟨u⟩(x−sv)ds.​

    Similar to the previous derivation, we let f:=⟨u−w⟩, then integrate the above equation over the whole Sd−1, it shows (using the lemma, polar coordinate transformation, maximum principle)

    ∣f(x)∣​≤C∫V​∫0τ−​(x,v)​∣f(x−tv)∣dtdv+νd−1​1​∫D​∣x−y∣d−1e−σs​∣x−y∣​σs​∣f(y)∣dy+∫Sd−1​∫0τ−​(x,v)​∫0s​e−sσs​L∣f(x−tv)∣σs​⟨u⟩(x−sv)dtdsdv≤C∫V​∫0τ−​(x,v)​∣f(x−tv)∣dtdv+νd−1​1+Lℓh​∫D​∣x−y∣d−1e−σs​∣x−y∣​σs​∣f(y)∣dy,​

    where ℓ=diam(D). We make the following observation.

    Commutativity Lemma

    Let A and S be L2(D)↦L2(D) operators,

    Af:=∫V​∫0τ−​(x,v)​∣f(x−tv)∣dtdv,Sf:=∫D​∣x−y∣d−1e−σs​∣x−y∣​f(y)dy.

    Then A commutes with S.

    A quick proof for this property. Let f be extended to Rd with zero outside D, then we can write

    SAf=∫Rd​∫V​∫0∞​∣x−y∣d−1e−σs​∣x−y∣​f(y−sv)dsdvdy,

    and we can derive AS similarly:

    ASf=∫V​∫0∞​∫Rd​∣x−sv−y∣d−1e−σs​∣x−sv−y∣​f(y)dydsdv=∫V​∫0∞​∫Rd​∣x−y′∣d−1e−σs​∣x−sv−(y′−sv)∣​f(y′−sv)dy′dsdv.

    That implies the following relation.

    ∣f(x)∣≤CA∣f∣+νd−1​(1+Lℓh)​σs​S∣f∣.

    Then we have the concluding theorem by noticing that ∥S∥op​≤1−e−σs​ℓ (why?).

    Uniqueness Theorem

    If (1+Lℓh)(1−e−σs​ℓ)<1, then the cone-beam source permits a unique solution (if exists).

    Notes

    • This uniqueness result simply serves as an exercise utilizing the Grönwall’s inequality. This result is only feasible for weak scattering medium.

    • The commutativity is necessary to estimate the spectral radius, a slightly more general condition is mentioned in2.
